# AlexNet CIFAR-100 图片分类使用指南

## 📁 文件说明

- `train.py` - 训练脚本
- `test_best_model.py` - 测试最佳模型的准确率
- `predict_image.py` - **对单张图片进行分类**（这是你需要的）
- `demo_save_test_image.py` - 从测试集提取示例图片

## 🚀 快速开始：单张图片分类

### 方法1：使用示例图片测试

```bash
# 1. 先生成一些示例图片
python demo_save_test_image.py

# 2. 对示例图片进行分类
python predict_image.py --image demo_images/test_1_*.png
```

### 方法2：使用你自己的图片

```bash
# 对单张图片分类（显示Top-5结果）
python predict_image.py --image /path/to/your/image.png

# 显示Top-10结果
python predict_image.py --image /path/to/your/image.png --top_k 10

# 使用不同的模型checkpoint
python predict_image.py --image your_image.jpg --model ./checkpoints/cifar100_epoch_45.pth
```

## 📊 支持的图片格式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- BMP (.bmp)
- 其他PIL支持的格式

**注意**：
- 图片会自动缩放到32x32（CIFAR-100的标准尺寸）
- 任何尺寸的图片都可以输入

## 💡 使用技巧

1. **图片质量**: 清晰的、主体明显的图片效果更好
2. **图片内容**: 应该包含CIFAR-100的100个类别之一
3. **置信度解读**:
   - > 50%: 高置信度，预测很可能正确
   - 30-50%: 中等置信度，有一定把握
   - < 30%: 低置信度，可能是模型不确定或不在训练类别中

4. **如果预测不准**:
   - CIFAR-100的训练图片是32x32的低分辨率图片
   - 模型在高分辨率或复杂背景的图片上可能表现不佳
   - 某些相似类别（如不同的猫科动物）容易混淆

## 🔄 继续训练

如果想提高准确率，可以继续训练：

```bash
# 从最佳模型继续训练
python train.py --phase train --resume ./checkpoints/cifar100_best.pth --epochs 100
```

## 📞 常见问题

**Q: 为什么我的猫/狗图片识别错误？**
A: CIFAR-100没有"cat"或"dog"这两个类别，只有更具体的分类（如tiger, leopard等）

**Q: 可以用这个模型识别其他物体吗？**
A: 只能识别CIFAR-100的100个类别中的物体

**Q: 为什么预测这么慢？**
A: CPU训练/推理较慢，如果有GPU会快得多

**Q: 如何提高准确率？**
A: 1) 继续训练更多epoch; 2) 调整学习率; 3) 使用数据增强; 4) 尝试更现代的架构（如ResNet）
